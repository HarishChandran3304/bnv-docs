---
title: bnv add
---

## Usage

```bash
bnv add KEY1 [KEY2 KEY3 ...]
```

## Description

Add one or more key names to the current project’s `.better-env` so they’ll be loaded.

### Examples

```bash
# Add a single key
bnv add DATABASE_URL

# Add multiple keys at once
bnv add DATABASE_URL API_KEY STRIPE_SECRET
```
