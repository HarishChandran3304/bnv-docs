---
title: bnv copy
---

## Usage

```bash
bnv copy KEY [KEY2 KEY3 ...]
```

## Description

Decrypt and copy secret value(s) to the clipboard. Values are not printed to the terminal.

### Examples

```bash
# Copy a single key's value to clipboard
bnv copy DATABASE_URL

# Copy multiple values
bnv copy API_KEY STRIPE_SECRET
```
